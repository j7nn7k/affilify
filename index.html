jQuery Affilify Demo

Demo Links

Affilify will transform product links to affiliate links automatically. Open the browser's web tools and check the network tab for the redirect when you click on the links. The first request in the network tab is the request to the affiliate network. Usually the affiliate networks will then redirect you to the original location of the link. Here's is a sample configuration: 

            // init affilify plugin
            $("a").affilify({
                zanoxPublisherId: "36434335C584445997",
                amazonPublisherId: "f07e5-21",
                affilinet: {
                    publisherId: "756180",
                    programs: [
                        // affilinet programs
                        {domain: "decathlon.de", siteId: "10685"},
                        {domain: "fitmart.de", siteId: "12802"},
                        {domain: "sportscheck.com", siteId: "3617"}
                    ]
                }
            });
